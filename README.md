## iOS Weather APP

- TabView Navigation.
- Works based on the user location also loads 3 cities (Buenos Aires, Montevideo, London).
- Displays Min/Max temperatures and sunrise and sunset at each city local time.
- Supports both Light Mode and Dark Mode configurations.
- Includes basic Unit Tests.
- Config file with key not included in this repo for obvious reasons.
